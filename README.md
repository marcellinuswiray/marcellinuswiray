## About Me

I'm a **systems programmer** based in **Indonesia**, focused on blockchain technology and cross-platform build engineering. I work with privacy-focused cryptocurrency projects, building and maintaining multi-platform compilation toolchains for complex C/C++ codebases.

```yaml
name: Marcellinus Wiray
location: Indonesia
role: Systems Programmer & Blockchain Developer
focus:
  - Privacy-focused cryptocurrency development
  - Cross-platform build systems & toolchain engineering
  - Low-level systems programming (C/C++)
  - Deterministic & reproducible builds
philosophy: >
  "The best code is the code that never has to be debugged
   — but let's be honest, that code doesn't exist."
```

### What I Do

- Building & maintaining **multi-platform compilation toolchains** for complex C/C++ codebases
- Contributing to **Sumokoin** — a privacy-focused cryptocurrency forked from Monero/CryptoNote
- Designing **cross-platform build systems** targeting **Linux, macOS, Windows, Android, and iOS**
- Engineering **CMake toolchains**, **Makefile automation**, and **multi-architecture compilation** (x86-64, ARM v6/v7/v8, i686)
- Advocating for **code reproducibility**, **deterministic builds**, and **open-source development**

### What I'm Exploring

- Advanced **zero-knowledge proofs** and their applications in privacy tech
- **Rust** for systems programming and blockchain development
- **WebAssembly** for cross-platform runtime environments
- Improving **CI/CD pipelines** for multi-arch builds

## Featured Project

### Sumokoin Build System

> Multi-platform build configuration for the **Sumokoin** privacy-focused cryptocurrency, based on the Monero codebase.

<table>
<tr>
<td width="50%">

**Build Features**
- **C++17 / C11** standards
- Security-hardened compilation flags (`-fstack-protector`, PIE, RELRO)
- Static & dynamic linking support
- Memory availability checks (3.5GB min)
- ccache integration for faster rebuilds

</td>
<td width="50%">

**Platform Coverage**
- Linux (multiple distros)
- macOS (Xcode SDK)
- Windows (MSYS2/MinGW)
- Android (NDK standalone toolchain)
- iOS (Xcode SDK)
- FreeBSD

</td>
</tr>
</table>

**Architecture Support Matrix:**

```
  x86-64  ........  Linux | macOS | Windows
  i686    ........  Linux | Windows
  ARMv6   ........  Linux (Raspberry Pi)
  ARMv7   ........  Linux | Android
  ARMv8   ........  Linux | Android | iOS
```

**Build Targets Available:**

```makefile
release / debug          # Standard builds
release-static           # Fully static binary
release-android          # Android NDK cross-compile
release-win64 / win32    # Windows via MSYS2
release-linux-armv6/7/8  # ARM cross-compilation
release-freebsd          # FreeBSD support
test / fuzz              # Testing & fuzzing (AFL)
coverage                 # Code coverage reports
```
